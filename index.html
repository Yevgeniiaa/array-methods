Методи масивів

Масив - це сукупність даних однакового або різного типу, які можна обробити як одне ціле. Ми можемо змінювати, додавати, вилучати елементи з масиву Приклади масивів: let arr = [1,2,3,4,5]

Назва методу
Length
Що робить метод?
Нічого
Що повертає?
Повертає довжину масиву
Які параметри має?
Немає параметрів і дужок
let arr = [1,2,3,4,5]
arr.length
Назва методу
toString ()
Що робить метод?
Дозволяє зробити об'єкти більш зрозумілими та інформативними
Що повертає?
Повертає значення типу String, яке представляє об'єкт у текстовій формі
Які параметри має?
Не приймає жодних параметрів
let arr = [1,2,3,4,5]
arr.toString()
Назва методу
join ()
Що робить метод?
З'єднує елементи масиву або колекції в єдиний рядок, вставляючи між ними роздільник
Що повертає?
Повертає об'єкт типу String, що містить об'єднані елементи з вказаним роздільником між ними
Які параметри має?
delimiter — роздільник, який буде додано між кожним елементом. elements — масив елементів типу CharSequence
let arr = [1,2,3,4,5]
arr.join()

Що повертає?

Які параметри має?
Не має параметрів
let arr = [1,2,3,4,5]
arr.pop()

unshift ()

Вставляє вказані елементи на початок масиву
Що повертає?
Повертає довжину нового масиву

value, start, end
let arr = [1,2,3,4,5]
arr.unshift ()
Назва методу
splice ()
Що робить метод?
Видаляє елементи з масиву, додає нові елементи у вказане місце, замінює певні елементи новими
Що повертає?
Повертає масив, що містить видалені елементи. Якщо нічого не видалено, повертає порожній масив
Які параметри має?
start, deleteCount (необов'язковий), item1, item2, ..., itemN (необов'язкові)
let arr = [1,2,3,4,5]
arr.splice ()
Назва методу
concat ()
Що робить метод?
Обʼєднує два або більше масивів або додавання елементів до масиву, створюючи новий масив із поєднанням всіх значень
Що повертає?
Повертає новий масив, що складається з елементів вихідного масиву, а також із доданих значень
Які параметри має?
value1, value2, ..., valueN
let arr = [1,2,3,4,5]
arr.concat ()
Назва методу
slice ()
Що робить метод?
Копіює частину масиву, визначену початковим і кінцевим індексами, у новий масив
Що повертає?
Повертає новий масив, який містить елементи, скопійовані з вказаного діапазону оригінального масиву
Які параметри має?
begin, end
let arr = [1,2,3,4,5]
arr.slice ()
Назва методу
sort ()
Що робить метод?
Сортує елементи масиву в місці, тобто змінює початковий масив
Що повертає?
Повертає відсортований масив (той же масив, на якому викликано метод)
Які параметри має?
compareFunction (необов'язковий) — функція порівняння, яка визначає порядок сортування
let arr = [1,2,3,4,5]
arr.sort ()
Назва методу
reverse ()
Що робить метод?
Перевертає порядок елементів у масиві: перший елемент стає останнім, другий — передостаннім
Що повертає?
Повертає відсортований у зворотному порядку масив (той самий масив, на якому викликано метод)
Які параметри має?
Не має параметрів
let arr = [1,2,3,4,5]
arr.reserve ()
Назва методу
random ()
Що робить метод?
Генерує випадкове число з плаваючою комою у діапазоні від 0 (включно) до 1 (виключно)
Що повертає?
Повертає випадкове число з плаваючою комою у діапазоні від 0 (включно) до 1 (виключно)
Які параметри має?
Не має параметрів
let arr = [1,2,3,4,5]
arr.random ()
Назва методу
copyWithin ()
Що робить метод?
Копіює частину масиву, визначену діапазоном індексів, і вставляє її на іншу позицію в межах того ж масиву
Що повертає?
Повертає модифікований масив (той самий масив, на якому викликано метод), зі зміненим порядком елементів
Які параметри має?
target, start, end
let arr = [1,2,3,4,5]
arr.copyWithin ()
Назва методу
every ()
Що робить метод?
Перевіряє, чи всі елементи масиву проходять перевірку, яку визначає функція зворотного виклику
Що повертає?
Повертає true, якщо всі елементи масиву проходять перевірку функції зворотного виклику. false, якщо принаймні один з елементів не проходить перевірку.
Які параметри має?
target, element, index, array, thisArg (необов'язковий)
let arr = [1,2,3,4,5]
arr.every ()
Назва методу
fill ()
Що робить метод?
Заповнює масив значенням, яке ви вказали
Що повертає?
Повертає модифікований масив (той самий масив, на якому викликано метод), зі зміненими значеннями
Які параметри має?
value, start, end
let arr = [1,2,3,4,5]
arr.fill ()
Назва методу
filter ()
Що робить метод?
Перебирає елементи масиву і перевіряє їх на відповідність умові, що задається функцією зворотного виклику
Що повертає?
Повертає новий масив, що містить елементи, які відповідають умові, заданій у функції зворотного виклику
Які параметри має?
callback, element, index, array, thisArg
let arr = [1,2,3,4,5]
arr.filter ()
Назва методу
find ()
Що робить метод?
Перебирає елементи масиву і застосовує функцію зворотного виклику до кожного елемента
Що повертає?
Повертає Перший елемент, який задовольняє умову, задану у функції зворотного виклику. undefined, якщо жоден елемент не відповідає умові
Які параметри має?
callback, element, index, array, thisArg
let arr = [1,2,3,4,5]
arr.find ()
Назва методу
findIndex ()
Що робить метод?
Перебирає елементи масиву і застосовує до кожного з них функцію зворотного виклику, щоб перевірити, чи відповідає елемент умові
Що повертає?
Повертає індекс першого елемента, який відповідає умові. -1, якщо жоден елемент не задовольняє умову
Які параметри має?
callback, element, index, array, thisArg
let arr = [1,2,3,4,5]
arr.findIndex ()
Назва методу
forEach ()
Що робить метод?
Виконує побічні ефекти (наприклад, модифікація зовнішніх змінних або виведення результатів)
Що повертає?
Не повертає значення. Він завжди повертає undefined
Які параметри має?
callback (обов'язковий), thisArg (необов'язковий)
let arr = [1,2,3,4,5]
arr.forEach ()

indexOf ()

Повертає індекс першого входження елемента в масив або рядок
Що повертає?
Повертає індекс першого входження елемента в масив або рядок
Які параметри має?
let arr = [1,2,3,4,5]
arr.indexOf ()
Назва методу
isArray ()
Що робить метод?
Метод дозволяє точно перевірити, чи є дане значення масивом
Що повертає?
Повертає true, якщо це масив, і false в іншому випадку
Які параметри має?
value: значення, яке потрібно перевірити
let arr = [1,2,3,4,5]
arr.isArray ()
Назва методу
lastIndexOf ()
Що робить метод?
Перевіряє чи є значення масивом
Що повертає?
Повертає true, якщо передане значення є масивом. false, якщо передане значення не є масивом
Які параметри має?
value
let arr = [1,2,3,4,5]
arr.lastIndexOf ()
Назва методу
reduceRight ()
Що робить метод?
Виконує зворотний обхід масиву (починаючи з останнього елемента і рухаючись до першого)
Що повертає?
Повертає одне значення — результат обробки всіх елементів масиву, яке накопичується через функцію зворотного виклику callback
Які параметри має?
callback (обов'язковий), initialValue (необов'язковий)
let arr = [1,2,3,4,5]
arr.reduceRight ()
Назва методу
reverse ()
Що робить метод?
Змінює порядок елементів масиву на зворотний, тобто перший елемент стає останнім, другий — передостаннім
Що повертає?
Повертає сам масив, але вже з елементами у зворотному порядку
Які параметри має?
Не має параметрів
let arr = [1,2,3,4,5]
arr.reverse ()
Назва методу
some ()
Що робить метод?
Перевіряє, чи хоча б один елемент масиву задовольняє умові, визначеній у функції зворотного виклику
Що повертає?
Повертає true, якщо хоча б один елемент масиву задовольняє умові (якщо функція зворотного виклику для хоча б одного елемента повертає true). false, якщо жоден елемент не задовольняє умові (якщо для всіх елементів функція зворотного виклику повертає false)
Які параметри має?
callback (обов'язковий), thisArg (необов'язковий)
let arr = [1,2,3,4,5]
arr.some ()
Назва методу
valueOf ()
Що робить метод?
Метод є вбудованим методом об'єктів, який використовується для отримання примітивного значення об'єкта
Що повертає?
Повертає примітивне значення, яке відповідає об'єкту. Як саме визначається це значення, залежить від типу об'єкта
Які параметри має?
Не має параметрів
let arr = [1,2,3,4,5]
arr.valueOf ()
